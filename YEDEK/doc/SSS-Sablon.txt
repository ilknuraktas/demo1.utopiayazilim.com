Şablonlar Sıkça Sorulan Sorular



S - Şablon dosyalarına nereden ulaşabilirim?

C - Kullandığınız şablon (include/conf.php dosyasında $aktifTemplateDizini değeri) ile ilgili tüm dosyalar templates/<aktifTemplateDizini>/ dizini içerisindedir.



S - Şablon klasör yapısı nasıldır?

C - Her şablon klasöründe aşağıdaki dizin ve klasörler bulunmaktadır.

	blocks <dizin> : Şablon dosyalarında kullanılan blocklara ait dış HTML templatelerdir. Blocklara ait {%BASLIK%} ve {%ICERIK%} olmak üzere iki makro kullanılabilir.

	images <dizin> : Şablona ait resim dosyalarını içerir.

	lists <dizin> : Tekrarlanan listelemeleri oluşturan şablonlardır. 

	systemDefault <dizin> : Varsayılan olarak kullanılan HTML şablonlara ait dosyaları içerir.

	systemDefault/urunGoster.php <dosya> : Ürün detay ekranında kullanılan HTML şablondur. PHP kod kullanılamaz ama {%FUNC_<fonksiyonadi>%} ile PHP fonksiyonlar çagırılabilir. Ör : {%FUNC_test%} yazıldıgı zaman makro test() fonksiyonunun dondüğü değer ile değiştirilir.

	systemDefault/urunList.php <dosya> : Genel ürün listeleme (tekrarlama) HTML şablonudur. 

	systemDefault/urunListShow.php <dosya> : Genel ürün listeleme (ürün gösterim) HTML şablonudur.

	systemDefault/urunListPager.php <dosya> : Ürün listeleme sayfalarında, sayfalama yapan PHP fonksiyonun değerlerini aldığı HTML şablondur.

	config.xml <dosya> : Şablona ait genel ayar dosyasıdır. Bu dosya ile şablondaki resim boyutlarını, sütun sayısını,şablonlarda blocklarında çağrılacak şablon dosya isimlerini set edebilirsiniz.

	temp.php <dosya> : Genel şablon dosyasıdır. PHP kod kullanılabilir. $PAGE_OUT yazan yerde ilgili sayfaya ait çıktı yer alır. Şablona ait hem header hemde footer bölümlerini bu dosya üzerinden set edebilirsiniz.

	index.php <dosya> : Sadece anasayfada ekrana çağrılan ve temp.php'de $PAGE_OUT kısmıne gelecek olan dosyadır.

	lib.php <dosya> : Şablona özel PHP fonksiyonları içerir.

	style.css <dosya> : Şablona özel CSS bilgilerini içerir.

	temp.js <dosya> : Şablona özel Javascript fonksiyonları içerir.



S - Ürün gösterim şablonlarında hangi hazır makroları kullanabilirim?

C - http://sorular.utopiayazilim.com/page.php?act=kategoriGoster&katID=325&autoOpen=16



S - Fiyatları KDV Dahil/KDV Hariç olarak göstermek istiyorum.

C - Bir önceki sorudaki makroları kullanarak fiyatları dilediğiniz gibi (KDV Dahil, hariç, YTL ..) gösterimini yapabilirsiniz.



S - E-Postalarımı oluşturan şablon dosyası nerededir?

C - templates/system/email/temp.php dosyasından ilgili şablonu güncelleyebilirsiniz.



S - Yönetim panelini olulturan şablon dosyası nerededir?

C - Yönetim paneli şablon dosyalarını templates/system/admin klasöründe bulabilirsiniz.



S - Logomu nereye upload etmem gerekiyor?

C - Logonuzu FTP üzerinden templates/<aktif sablon dizini>/images/ klasörüne kopyalamanız yeterlidir. templates/<aktif sablon dizini>/images/temp.php dosyasından logo img tag'ini düzenleyebilirsiniz.



S - Flash reklam banner'ında upload ettiğim resimler çıkmyor.

C - Flash banner kullanan templatelerde (green,clean,hean) resim upload ederken .jpg uzantısı oldugundan emin olun. Upload ettiğiniz resim .jpg uzantılıysa ana dizindeki gallery_data.xml dosyasına ve images/kampanya klasorune yazma hakkı verildiğiniz kontrol edin.



S - Flash reklam bolumunde upload ettiğim resimler sığmıyor.

C - İki yerden flash banner enini ve boyutunu arttırmanız gerekiyor.

    1. templates/<kullandığınız template dizini>/index.php

    2. Yönetim Paneli->Reklam->Kampanya Flash Ayarları 



S - Ürün resim boyutlarını nereden değiştirebilirim?

C - templates/[Kullandığınız template dizini]/config.xml dosyasından otomatik küçültülen resim boyutlarını değiştirebilirsiniz.



S - Şablonuma banner nasil eklerim?

C - 

	1) Reklam yonetimi bolumunden ekleyeceginiz firmayi ekleyin. (Firmalar -> Yeni Ekle)

	2) Eklediginiz firmaya ait banner bilgilerini ekleyin. (Banner Bilgileri -> Yeni Ekle)

	3) Son olarak banner yonetimi bolumunden hazirladiginiz bannerin limitlerini belirleyip ona bir kod vererek kaydedin. Kaydettiginiz zaman size 



"Banner'ı siteye eklemek için template dosyanıza (temp.php) <?=insertBanner('BANNER-KODU')?> kodunu ekleyin"



mesaji gelecektir. Orada yazdigi gibi temp.php dosyaniza istediginiz yere <?=insertBanner('BANNER-KODU')?> yazip kaydedikten sonra eklediginiz yerde banner gozukmeye baslayacaktir. Istatistiklere reklam yonetimi bolumunde bulunan "Banner Istatistikleri" bolumunden kontrol edebilirsiniz.
